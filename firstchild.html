the given css styling property will be auto matically added.

In this property if the given-element is the last child of it's parent then

GETTING CHILD COUNT OF THE PARENT

documnet.getElementById("idName").childElementCount.



In this property it returns the number of child elements for the element
whose id is denoted.
The childElementCount property will produce the same result as element.children.length.
GETTING CHILD NODES OF THE PARENT

document.element.childNodes.



In this property it returns as a Nodelist of child Nodes for the given parent element.
It appears as sorted order as they are shown in the code.
so they can be accessed by using their index values.
GETTING CHILDREN NODES OF THE PARENT

document.element.children.



In this property it returns as a collection of child Nodes for the given parent element.
It appears as sorted order as they are shown in the code.
so they can be accessed by using their index values.
Almost, same as childNodes only difference child Nodes returns text and comment lines also but children return only elements.
GETTING First Child

document.element.firstChild.



It returns child elment of the given parent as a HTML object.
GETTING First Element Child

document.element.firstElementChild.



It returns child elment of the given parent as a HTML object.
Almost, same as first-child only difference first-child Nodes may also returns text and comment lines also but firstElementchildren return only elements.
GETTING Last Child

document.element.lastChild.



It returns last-child element of the given parent as a HTML object.
GETTING Last Element Child

document.element.lastElementChild.



It returns last-child element of the given parent as a HTML object.
Almost, same as last-child only difference last-child Nodes may also returns text and comment lines also but lastElementchildren return only elements.
GETTING Next Sibiling

document.element.nextSibiling.



It returns single element who is the next-sibling given element as a HTML object.
GETTING Previous Sibiling

document.element.npreviousSibiling.



It returns single element who is the previous-sibling given element as a HTML object based on same order of the code.
