Class List

Class List returns the class name of the elements.
This property is used to add,remove and toggle CSS classes on the elements.
It is a read only property it can be modified by using add and remove methods.

Multiple class can be given in add method and remove method.











		
element.add(className[,...])

				This property is used to add new css class property to the existing element.
				Ex:
				document.getElementById("content").classList.add('contentStyleWhite');

		
element.remove(className[,...])

				This property is used to delete the added css class property from the existing element.
				Ex:
				document.getElementById("content").classList.toggle('contentStyleBlack');

		
element.toggle(className[)

				This property is used to on and off the given style class.
				Ex:
				document.getElementById("content").classList.remove('contentStyleWhite');

		
element.replace(className1,className2)

				This property is used to replace one class by the another class.
				Ex:
				document.getElementById("content").classList.replace('contentStyleBlack','contentStyleWhite');

		
element.contains(className)

				This property returns boolean values about the existence of the class.
				Ex:
				document.getElementById("content").contains('contentStyleWhite');
