innerHTML and innerText in input Tags.







		Using innerHTML and innerText content cannot be added into the input elements.
		It can be added using only value.
		Ex:
			document.getElementById('idName').value = "some content";
		The value assigned will show same as that of text.
	






		Using innerHTML and innerText content can be added into the inputarea elements on reload.But Cannot add content after editing.
		It can be added using only value.
		Ex:
			document.getElementById('idName').value = "some content";
		The value assigned will show same as that of text.
