Adding and Deleting HTML Elements

		document.createElement(element);  -->  Creates new HTML element.
		document.removeChild(element);	  -->  Removes an HTML element.
		document.appendChild(element);	  -->  Adds an HTML Element.
		document.replaceChild(element);	  -->  Replace an HTML element.
		document.write(text);

	
Creating Element :

		Creating a button using document.createElement(element).
		It  creates new element and we can add it where ever necessary.
		First using document.createElement() create an for the required tag.
		Then add Attributes for the  created Element.
		For adding contents to the created node, create a text Node and write some contents to it.
		Then append the textNode to it.

Note : After clicking the button check the end of the page. It is added there.

	
Appending Element :

		Appending means adding an new node to the existing node.
		It adds it at last of the existing node.
		document.body.append(element).
			->It adds it at the end of the body element.

	
Replacing an Element :

		Replacing an Element instead of an other element.
		If given element doesn't exist error occurs.
		Same like deleting should perform this operation from it's parent node.
	
	
Deleting Element :

		 Using Delete element we can delete elements one by one.
		 First select the node/tag/id/class using any css selectors.
		 Similar to tree, if you want to delete a node you can delete only from parent node.
		 From parent check whether it has existing child nodes and continue deleting by using index position.
