Z-Index

	Z-index property always determines the stack order of an element.
	Same as that of stack, elements with higher index always becomes front.

	For defining z-index property for an element it should have position property.
	--> Relative, Absolute, fixed.

	Property values of z-index are: 
		-->number -- number value/index position of the stack,
		   initial -- , auto, inherit
