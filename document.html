Document Object Model

	Document object model is an cross-platform that is it can work on all different platforms.
	The way of accessing or changing document  in window is document object model.
	It treats HTML as an tree structure/hirerachy with nodes as objects.
	It can manipulate all the objects dynamically.

	When the web page is loaded browser creates the document object model which is an objectal orientation of the HTML page.
	It intracts with the javascript and document itself for creation of dynamic web pages.
	It can be used for style and creation new dynamic content.
	It can manipulate the HTML element objects.

	
document.all

	It returns all the html elements in array format.
	But storing the elemnts in the arrays is similar to that of hirerachy,
	Ex:array[0] - gives whole HTML Page, a[1] - whole head tag,a[2] - elements of head tag. 

	syntax : document.all

	
document.anchor

	It is used to get all the anchor elements in the documents.
	It returns the value as an array.
	
property --> document.anchor.length
 It gives number of anchor elements in the document.

	syntax : document.anchor

	
document.open()

	It used for creating new page.
	It creates new page by deleting the content of the old to which the new content is going to writen.
	It does not return anything.

	syntax : document.open()

	
document.cookie

	It is used to set cookie in the browser when specific condition occurs.
	Cookie are the txt document created in the local browser by the server. 

	syntax : document.cookie="cokiename=cookieValue;expiryDate"

	
docment.location

	It returns the location of current working page.
	Its return type is object.
	Using this we can get host name,path,protocol being used in this page.

	syntax :	document.location
			document.location.host
		  	document.location.path
			document.location.href
			document.location.protocol

	
document.body

	It returns < body> of the current document if no present returns null.
	This is editable changing the content to this would remove all the content of the body.

document.head

	It returns < head> of the current document.
	This is editable changing the content to this would remove all the content of the previous head. 

	syntax : document.head

	
document.styleSheets

	It returns external link of < styleSheets> of the current document.

	syntax : document.styleSheets
