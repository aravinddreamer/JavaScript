Storage

	Information can be stored in stored in web browser in different ways.

  1. Local storage
  2. Session storage
  3. Cookies


Local Storage :

	Local is that information are stores in local browser itself.
	It is not deleted until cleared or explicitly deleted.
	It's size is greater than that of cookie 5MB.
	It is stored in the client side.
	It reduces the traffic between the client and server.
	
It works on the same-origin policy that is only on the same page.


	It stores as key and value pair.
	It can be read,modified and deleted.
	Ex:

	key-value pair stored in google.co.in 
	will not be available google.co.in/mail.
	But if you open same page in different tabs it will be present.


	
localStorage.setItem(key,value)

		It is used to create key value pair at local storage.
		Ex:
		localStorage.setItem("userName","ARAVIND")

	
localStorage.getItem(key)

		It is used to get key value from the local storage.
		Ex:
		localStorage.getItem("userName")

	
localStorage.removeItem(key)

		It is used to remove  key value pair at local storage with the given key.
		Ex:
		localStorage.removeItem("userName")

	
localStorage.clear()

		It is used to clear all key value pair at local storage.
		Ex:
		localStorage.clear()

	
localStorage.key(value)

		It returns the key value of the given element if not present it returns null value.
		Ex:
		localStorage.key("ARAVIND")

	
localStorage.valueOf(key)

		It returns the key-value of the given element as a object if not present it returns null value.
		Ex:
		localStorage.valueOf("userName")
	
		

Session Storage :

	
It is same as local storage.
		But only one small difference.
		Only on that tab .Even same webpage on other tab it will not be present.
		When the tab is closed it will be deleted.
	
	
	
sessionStorage.setItem(key,value)

		It is used to create key value pair at local storage.
		Ex:
		sessionStorage.setItem("authKey","bSep23WDIX")

	
sessionStorage.getItem(key)

		It is used to get key value from the local storage.
		Ex:
		sessionStorage.getItem("authKey")

	
sessionStorage.removeItem(key)

		It is used to remove  key value pair at local storage with the given key.
		Ex:
		sessionStorage.removeItem("authKey")

	
sessionStorage.clear()

		It is used to clear all key value pair at local storage.
		Ex:
		sessionStorage.clear()

	
sessionStorage.key(value)

		It returns the key value of the given element if not present it returns null value.
		Ex:
		sessionStorage.key("bSEP23WDIX")

	
sessionStorage.valueOf(key)

		It returns the key-value of the given element as a object if not present it returns null value.
		Ex:
		sessionStorage.valueOf("authKey")
	

Cookies

	It is a small text document stores in the directory of the browser.
	It holds small space for holding information.
	It communicates with the server for every request.
	It can be set with expiry time when should be the cookie deleted.
	It is also key-value pair.
	It set using document.cookie="key=value;expiry".
	It is the convinenet way of carrying information from one session to the another.

	Mostly Cookies are for Server side reading and local storage, session storage are for client side reading as it rarely communicates with the server.
