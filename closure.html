Closure

Closure is mostly about the scope of the variable or an object. For Example: Function inside a Function.

Inner function can access the variables of the outer function above it. The variable declared the function is called as global variable.

When there are two variable inside the program. One outside the function block and another inside it.

Inner variable has more priority than that of outer variable. During, two values value of inner is used.

var val=10;

var val=15;

			function display(){

			}
			display();
		}

		document.getElementById('outerdisplay').innerHTML="< h2>Outer value : " + val+ "< /h2>";
