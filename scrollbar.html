OverFlow and Scroll Bar

Overflow : visible

In the visible property the content which exceeds the division area is visible out side the division ,contents flows out the division. No scroll bar occurs in this property.

Overflow : hidden

In the hidden property when the content which exceeds the division area it is hidden inside the division container. No scroll bar occurs in this property.

Overflow : auto

In the auto property when the content overflows from the division scroll bar appears automatically. Scroll is not added but occurs when content exceeds.

Overflow : scroll

In the scroll property Even when the content is within the limit also scroll bar occurs. Scroll is default in this property.It is always visible.

-webkit-scrollbar

is used to create an scroll when contents flow is more than the size of page.

-webkit-scrollbar-track

is used to add propeety for the scrollbar track.

-webkit-scrollbar-thumb

is used to add property for the thumb.

-webkit-scrollbar-button

to add button property.

-webkit-scrollbar-corner

to add property for corner of the scrollbar.
